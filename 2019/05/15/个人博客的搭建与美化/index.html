3：配置next主题

1.安装方式：

  • a.克隆next主题到目标目录

git clone https://github.com/iissnan/hexo-theme-next themes/next

  • b.next发布界面下载，然后移动到themes目录下。

2.启用主题：

与所有 Hexo 主题启用的模式一样。 当 克隆/下载 完成后，打开 站点配置文件， 找到 theme 字段，并将其值更改为 next。

theme: next

  1. 主题配置

基本配置包含：menu-菜单栏、social-社交、link-友情链接、font-字体等等设置，
具体修改方式参考 官方文档
